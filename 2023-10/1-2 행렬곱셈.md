행렬곱셈 구현하기


```python
def solution(arr1, arr2):
    answer = []
    
    m = len(arr1)
    n = len(arr2)
    p = len(arr2[0])
    
    for i in range(m):
        new_row = []
        for k in range(p):
            sum_ = 0
            for j in range(n):
                sum_ += arr1[i][j]*arr2[j][k]
            new_row.append(sum_)
        answer.append(new_row)    
    
    return answer
```
